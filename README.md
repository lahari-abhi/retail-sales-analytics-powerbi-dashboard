# 📊 Retail Sales Analytics Dashboard | Power BI

## 📌 Overview

This project presents an interactive Retail Sales Analytics Dashboard built using Microsoft Power BI. It demonstrates end-to-end data analysis, including data cleaning, data modeling, DAX calculations, KPI creation, and interactive visualizations.

---

## 🛠️ Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Microsoft Excel

---

## 📂 Dataset

The dashboard is built using a retail sales dataset containing:

- Sales Transactions
- Customers
- Products
- Regions
- Calendar

---

## 🚀 Key Features

- Data Cleaning using Power Query
- Star Schema Data Modeling
- DAX Measures and KPIs
- Interactive Slicers
- Cross-filtering Between Visuals
- Sales Trend Analysis
- Customer Analysis
- Product Performance Analysis
- Regional Sales Analysis

---

## 📊 Dashboard Features

- Total Sales
- Total Orders
- Total Customers
- Monthly Sales Trend
- Sales by Region
- Sales by Category
- Interactive Filtering
